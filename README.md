## About The Project

we built a automatic attendance detection project with the use of yolov5 by using our custom dataset

### Built With

* yolov5
* roboflow



<!-- GETTING STARTED -->
## Getting Started

we used google colab to run our yolov5 model to run it with gpu

### Prerequisites

* python
* roboflow

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ultralytics/yolov5.git
   ```
2. Install required dependensies
   ```sh
   pip install -r requirements.txt
   ```
4. Enter your roboflow API of your custom dataset
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
